About This Portfolio

What You'll Find Here

This ePortfolio was built as part of Assignment 1 for CET138. Rather than simply listing definitions, I wanted each section to reflect how my understanding of each technology grew during the trimester — the concepts I found challenging, the patterns that clicked, and the techniques I practised hands-on.

Every page in this portfolio is itself an example of what it describes: the HTML is semantic and accessible, the CSS is modular and responsive, and the JavaScript uses modern, performance-aware patterns.

  • Understand the full web development stack from frontend to backend
  • Build structured, semantic webpages using HTML5
  • Design responsive interfaces with modern CSS techniques
  • Accelerate UI development using the Bootstrap framework
  • Implement interactive behaviour and DOM manipulation with JavaScript
